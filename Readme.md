# AgentDesk

A Multi-Agent Customer Support System built using FastAPI, React, LangGraph, PostgreSQL, and ChromaDB.

## Features

* Multi-Agent Architecture
* Support Agent
* Knowledge Base Agent (RAG)
* Escalation Agent
* Conversation History
* Real-Time Admin Dashboard
* CI/CD with GitHub Actions

## Tech Stack

### Frontend

* React
* Vite

### Backend

* FastAPI
* PostgreSQL
* SQLAlchemy

### AI

* LangChain
* LangGraph
* ChromaDB

## Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/support-graph-ai.git
cd support-graph-ai
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
